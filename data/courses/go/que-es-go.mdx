Go, también conocido como Golang, es un lenguaje de programación desarrollado por Google. Está diseñado para ser simple, eficiente y fácil de aprender. 

Se utiliza a menudo para construir sistemas de gran escala, concurrentes y distribuidos. También, comúnmente para construir servidores web, servicios de red y otros tipos de software de back-end. 

Algunas de las características clave de Go incluyen la recolección de basura, el soporte incorporado de concurrencia y un sistema de tipos estricto.
